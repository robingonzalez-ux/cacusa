# CACUSA Store UI Kit

A high-fidelity, click-through prototype of the CACUSA online jewelry store.

## Screens
1. **Homepage** — Hero, featured collections, bestsellers grid
2. **Product Detail** — Personalization form, add to cart, gallery
3. **Cart / Checkout** — Order summary, shipping fields
4. **Collections** — Filter + product grid

## Usage
Open `index.html` to navigate the full store prototype.

## Components
- `Header` — Logo, nav, cart icon
- `HeroSection` — Watercolor gradient hero with tagline
- `ProductCard` — Product tile with hover, tags, pricing
- `ProductGrid` — Responsive 3-col grid
- `ProductDetail` — Full detail view with personalization
- `CartDrawer` — Slide-in cart panel
- `Footer` — Links, social, tagline

## Design Notes
- Fonts: Montserrat (display), Cormorant Garamond (body serif), DM Sans (UI)
- Colors: Rose #E8508A primary, Violet #B07FEC accent, Gold #C9A46A jewelry
- Motion: 300ms ease-out transitions; card lift on hover (translateY -4px)
